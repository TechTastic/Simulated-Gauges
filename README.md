# Simulated Gauges
#### A Deployer-based addon for Create: Simulated, best used with Extra Gauges!

Simulated Gauges uses Deployer to allow Logistics Gauges to get and utilize physics information from various Create: Simulated blocks!

List of supported Blocks:
- Altitude Sensor
    - This sensor can output its height in the world as a number for Deployer-based gauges.
- Velocity Sensor
    - This sensor can output its velocity in the world as a number for Deployer-based gauges.
- Navigation Table
    - This block can output its adjusted angle relative to its target destination as a number for Deployer-based gauges.
- Nameplate
    - This block can output its text as a string for Deployer-based gauges.
- Linked Typewriter
    - This block can output its held key's keycode as a number for Deployer-based gauges as well as output the typed string of the Typewriter as a string for Deployer-based gauges.

Future Content:
